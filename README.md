# 个人技术主页

基于 Vue3 + Spring Boot 的个人技术展示主页，用于展示个人信息、技术栈和 GitHub 项目。

## 项目说明

本项目是本科毕业设计作品，主要功能包括：

- 个人信息展示
- 技能雷达图（ECharts）
- 技术栈标签云
- 动态欢迎语
- GitHub 项目列表同步
- 响应式布局

## 技术栈

**前端**：Vue3、Element-Plus、ECharts、Axios、Vite

**后端**：Spring Boot、MySQL、Redis、MyBatis-Plus
